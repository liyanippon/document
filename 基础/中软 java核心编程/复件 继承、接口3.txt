2.	动物接口除具备生物接口特征之外，还定义了吃饭和睡觉两个抽象方法；　
3.	人接口除具备动物接口特征外，还定义了思维和学习两个抽象方法。　
定义一个学生类实现上述人接口。




模拟照相过程
步骤一 声明能拍照的工具类型 接口Photographable,并添加拍照方法 takePhoto()
步骤二 声明照相机类Camera 实现拍照接口
声明照手机类 Phone 实现拍照接口
步骤三 添加人类，并具有照相的功能，用不同的设备去照相


 1、 定义一个接口CanFly，描述会飞的方法public void fly();　
2、 分别定义类飞机和鸟，实现CanFly接口。　
3、定义一个测试类Test，测试飞机和鸟。测试类中定义一个makeFly()方法，让会飞的事物飞起来。　
4、然后在main方法中创建飞机对象和鸟对象，并在main方法中调用makeFly()方法，让飞机或鸟起飞



二 玻璃的选择
步骤一  添加代表透明度的接口 Transparency，在这个接口中定义了代表透明度取值的3个静态常量
		变量 OPAQUE=1 代表不透明
		变量 TRANSPARENT=2 代表透明
		变量 TRANSLUCENT=3 代表半透明
	  	并添加获得透明度的方法  int getTransparency();
步骤二  创建玻璃类  Glass 实现了Transparency 接口
步骤三  在测试类中创建两块玻璃，一块用做客厅窗户的透明玻璃，另一块用做卫生间空间的不秀明玻璃，
注意：利用接口获得常量的用法
